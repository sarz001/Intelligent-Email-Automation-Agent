# === Setup Instructions ===
# 1. Go to Google Cloud Console (https://console.cloud.google.com/).
# 2. Create a project and enable the Gmail API and Google Calendar API.
# 3. Configure OAuth 2.0 Client ID (Desktop app) and download credentials.json.
# 4. Place credentials.json in the project root directory.
# 5. On first run, the script will generate token.json after you authenticate.
# 6. Ensure that SCOPES includes:
#    - https://www.googleapis.com/auth/gmail.modify
#    - https://www.googleapis.com/auth/calendar.events
# 7. Authorize using the same Gmail account you want to access.
